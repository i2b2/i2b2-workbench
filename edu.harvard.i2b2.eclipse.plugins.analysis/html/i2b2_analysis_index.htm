Introduction

This section describes the layout and contents of this view.

 

Layout of the Analysis View

The analysis view is used display the analysis results of a query run via the query tool in the i2b2 Workbench.

 

Figure 1         Analysis View

 

 

The analysis view can be broken down into several sections which are defined below.

 

Graphic Analyses

The graphic analyses section of the analysis view contains the list of analyses that are associated to the previous query.

 

Figure 2         Graphic Analyses Section

 

 

Query Name

The name of the previous query that is being viewed will display in the Query Name field.

 

Figure 3         Query Name

 

 

Chart Section

The chart section contains the graph of the analysis result that is selected in the graphic analyses section.  At this time, all the charts that display in this area are a type of two-dimensional bar chart known as a category chart.

 

Figure 4         Chart Section

 

 

Chart Title

The title of the chart will display at the top of the chart section. The title that appears will always default to the type of analysis that is being viewed.

 

Note: The title can be temporarily modified by editing the chart properties.  

 

 

Domain Axis (x axis)

The domain axis (x axis) is the horizontal line of the chart.  This axis contains the concepts (categories) that are being analyzed.

 

Note: By default the orientation of the chart is vertical. If the orientation of the chart is change to horizontal the domain axis will be the vertical line of the chart. 

 

Figure 5         Domain axis - Vertical orientation

 

 

Figure 6         Domain axis – Horizontal orientation

 

 

Range Axis (Y Axis)

The range axis (y axis) is the vertical line of the chart and contains the numerical values (counts).

 

Note: By default the orientation of the chart is vertical. If the orientation of the chart is change to horizontal the range axis will be the horizontal line of the chart. 

 

Figure 7         Range axis - Vertical orientation

 

 

Figure 8         Range axis – Horizontal orientation

 

 

 

Analysis Types

There are several different analysis types available at this time.

 

Note: The labels that appear on the x-axis for each of these breakdowns is directly related to what is stored in the database.

 

Age Patient Breakdown

Provides a breakdown of the patients by their age

Gender Patient Breakdown

Provides a breakdown of the patients by their gender

Patient Count

Provides a total number of the patients that met your search criteria

Race Patient Breakdown

Provides a breakdown of the patients by their race.

Vital Status Patient Breakdown

Provides a breakdown of the patients by their vital status.

 

 

 

Generate and View Analysis Results

This section describes how to generate analysis results as well as how to view these results in the analysis view.

There is only one way to generate the analysis results and that is by running a query with the desired type of analysis checked off.  Once the query has been run successfully there are two ways of viewing these results in the analysis view.

 

1.       Running an analysis query – the results will automatically display in the analysis view.

2.       Drag a previous query () from either previous query or workplace views.

 

 

Running a Query

This section describes the process of running a query as it pertains to gathering analysis results. A more detailed explanation of running queries can be found in the on-line help for the i2b2 Query Tool View.

Once the query has run and completed successfully, the workbench will automatically display the analysis results in the analysis view.

 

Output Selection

In the query tool users can select any combination of the analysis types explained below.

 

Figure 9         Analysis Types in Query Tool View

 

 

Patient Count

This option will return the total number of patients who met the criteria defined in the creation of the query.

 

Gender Patient Breakdown

This option will return a breakdown of the patients by their gender.  The patients returned are still dependent on the criteria defined in the creation of the query.

 

Vital Status Patient Breakdown

This option will return a breakdown of the patients by their vital status.  The patients returned are still dependent on the criteria defined in the creation of the query.

 

Race Patient Breakdown

This option will return a breakdown of the patients by their race.  The patients returned are still dependent on the criteria defined in the creation of the query.

 

Age Patient Breakdown

This option will return a breakdown of the patients by their age.  The patients returned are still dependent on the criteria defined in the creation of the query.

 

 

Add Previous Query

A previous query can be added to the analysis view by simply dragging the query name from one from either the previous query or workplace views.  The sections below will outline the process of adding a previous query from either of those views.

 

Previous Query View

The previous query view displays the queries that were run by the user. 

 

Figure 10       Previous Query View

 

 

Add Previous Query From Previous Query View

1.       Click on the previous query and drag the item over to the Analysis View.

2.       Drop the previous query into the Graphic Analyses section.

 

Figure 11       Dropping Previous Query

 

 

3.       The analyses associated with the previous query will display in the list and the name of the query will appear at Query Name.

 

Figure 12       Previous Query Added

 

 

Workplace View

Information in the workplace is related to the most common concepts and queries that an individual uses and in essence becomes their personal workplace.

 

Figure 13       Workplace View

 

 

Add Previous Query from Workplace View

1.       Find the previous query you want to add (either in your folder or a shared folder).

2.       Click on the previous query and drag it over to the Analysis View.

3.       Drop the previous query into the Graphic Analyses Section (see Figure 11).

4.       The analyses associated with the previous query will display in the list and the name of the query will appear at Query Name (see Figure 12).

 

 

 

Navigation

This section documents how to maneuver around the analysis view.  It covers topics such as clearing the view, zooming in and out of the results and saving an image of the chart.

 

Clear Graphic Analyses

Currently there is only one way to remove an analysis from the Analysis View and that is to clear the graphic analyses panel.

 

Clear Items in Panel

1.       Click on the delete button () located in the top right corner of the panel.

2.       The list of graphic analyses will be cleared.

 

Warning: Once you click on the X (delete button), all previous queries and their associated analyses will be removed; therefore you need to make sure you want to delete all items.

 

 

Zoom In/Out

The ability to zoom in and out is only available for the Range axis.

 

Zoom In

Users can zoom in by using either the mouse or selecting the option from the pop-up menu.

Mouse: Use the Left Mouse Button to zoom in on the view.  The point at which the button is pressed to the point at which it is released will define the range that is used to zoom in on the data.

 

Figure 14       Zoom In (Mouse)

 

 

Menu: Use the Right Mouse Button to pop-up the menu in the chart section.  Selecting Zoom In > Range Axis will automatically zoom the chart to the range defined in the properties settings.

 

Figure 15       Zoom In (Menu)

 

 

Zoom Out

Users can only zoom out by selecting the zoom out option from the pop-up menu.

 

Menu: Use the Right Mouse Button to pop-up the menu in the chart section.  Selecting Zoom Out > Range Axis will automatically zoom the chart out to the range defined in the properties settings.

 

Figure 16       Zoom Out (Menu)

 

 

Return to Default

Returning the chart to the default settings can be done by selecting Auto Range > Range Axis from the pop-up menu in the chart section.

 

Figure 17       Auto Range

 

 

Save Image

An image of the bar chart can be saved to a local file on your computer.  Images will be saved as a type of file called Portable Network Graphics (PNG).

 

1.       Using the Right Mouse Button click on the chart you want to save.

2.       Select Save As… from the pop-up menu.

 

Figure 18       Pop-up Menu

 

 

3.       The save dialog box will open.

 

Figure 19       Save Dialog Box

 

 

4.       Select the location you want to save the image to.

5.       Enter a name for the file at the File Name field.

6.       Click on the Save button.

 

 

Chart Properties

The properties allow the user to change the general appearance and layout of the chart.  Any changes made to the properties are temporary and are not stored with the analysis results.  Changing the properties is useful if you want to make changes to the appearance and then save an image of the chart.

 

Title

The title tab allows you to change information about the title that appears above the chart.  By default the title is the same as the analysis that was performed.

 

Figure 20       Properties: Title Tab

 

 

General

 

Figure 21       Properties: General Section of Title Tab

 

 

§         The Show Title checkbox determines whether or not a chart title will appear above the chart.  A checkmark in the checkbox means a title will appear.

§         The Text field is where you can enter the title you want to display.  The default title will be the type of analysis that was performed.

§         The Font field contains the type, style and size of the font used for the title. 

o        To change the font settings click on the Select button.  A new window will open allowing you to change the settings.

§         The Color field defines the color of the text in the title. 

o        To change the color click on the Select button.  A new window will open allowing you to select a new color.

 

 

Plot

The plot properties are grouped into three areas; (1) Domain Axis, (2) Range Axis and (3) Appearance. 

 

Domain Axis

The Domain Axis tab allows users to define how information will appear in relation to the domain axis (x axis).

 

Figure 22       Domain Axis Properties

 

 

General

 

Figure 23       Domain Axis Properties - General

 

 

§         The Label field is where you can enter the axis title you want to display.   By default this field is left blank.

§         The Font field contains the type, style and size of the font used for the label. 

o        To change the font settings click on the Select button.  A new window will open allowing you to change the settings.

§         The Paint field defines the color of the text in the label. 

o        To change the color click on the Select button.  A new window will open allowing you to select a new color.

 

 

Other

 

Figure 24       Domain Axis Properties - Ticks

 

 

Ticks

§         The Show tick labels checkbox determines whether or not the labels (categories) will display along the domain axis.  A checkmark in the checkbox means the labels will appear.

§         The Tick label font field contains the type, style and size of the font used for the label. 

o        To change the font settings click on the Select button.  A new window will open allowing you to change the settings.

§         The Show tick marks checkbox determines whether or not tick marks will display in the chart.  A checkmark in the checkbox means the tick mark will appear.

 

Note: At this time the only type of chart supported are category charts and the ability to show tick marks is not applicable to these types of charts.

§         The Font field contains the type, style and size of the font used for the label. 

o        To change the font settings click on the Select button.  A new window will open allowing you to change the settings.

§         The Paint field defines the color of the text in the label. 

o        To change the color click on the Select button.  A new window will open allowing you to select a new color.

 

 

Other

 

Ticks

 

Figure 27       Range Axis Properties - Ticks

 

 

§         The Show tick labels checkbox determines whether or not the labels (categories) will display along the domain axis.  A checkmark in the checkbox means the labels will appear.

§         The Tick label font field contains the type, style and size of the font used for the label. 

o        To change the font settings click on the Select button.  A new window will open allowing you to change the settings.

Range

 

Figure 28       Range Axis Properties - Range

 

 

§         The Auto-adjust range checkbox defines whether or not the axis will automatically adjusts its range to reflect the range of data values.  A checkmark in the checkbox means the range will be automatically adjusted.

§         The Minimum range value field defines the lower boundary for the axis.

§         The Maximum range value field defines the upper boundary for the axis.

 

 

Note: The minimum and maximum range value fields are only available if the auto-adjust range field is unchecked. 

 

 

Appearance

The Appearance tab allows users to define how the plot area of the chart will appear.

 

Figure 29       Appearance Properties

 

 

General

§         The Outline stroke field defines the width of the lines that form the outline of the plot area.

§         The Outline paint field defines the color of the lines that form the outline of the plot area.

§         The Background paint field defines the color to use in the background of the plot area.

§         The Orientation field defines the direction of the bar chart. 

o        The choices are either vertical or horizontal.

 

 

Other

The Other tab allows you to change general information about the charts appearance.

 

Figure 30       Other Properties

 

 

General

§         The Draw anti-aliased checkbox defines whether or not to use anti-aliasing to draw smooth looking charts.  A checkmark in the checkbox means the charts are drawn with anti-aliasing.

§         The Background paint field defines the color to use in the background of the chart area.

 

 

Tooltips

Hovering over a bar in the chart area will display the category name and value.
