i2b2 Analysis View

The analysis view is designed to provide the visualization of analysis results of queries that users run using the i2b2 Workbench.

Introduction

This section describes the layout and contents of this view.

 

Layout of the Analysis View

The analysis view is used display the analysis results of a query run via the query tool in the i2b2 Workbench.

 

Figure 1         Analysis View

 

 

The analysis view can be broken down into several sections which are defined below.

 

Graphic Analyses

The graphic analyses section of the analysis view contains the list of analyses that are associated to the previous query.

 

Figure 2         Graphic Analyses Section

 

 

Query Name

The name of the previous query that is being viewed will display in the Query Name field.

 

Figure 3         Query Name

 

 

Chart Section

The chart section contains the graph of the analysis result that is selected in the graphic analyses section.  At this time, all the charts that display in this area are a type of two-dimensional bar chart known as a category chart.

 

Figure 4         Chart Section

 

 

Chart Title

The title of the chart will display at the top of the chart section. The title that appears will always default to the type of analysis that is being viewed.

 

Note: The title can be temporarily modified by editing the chart properties. 

 

 

Domain Axis (x axis)

The domain axis (x axis) is the horizontal line of the chart.  This axis contains the concepts (categories) that are being analyzed.

 

Note: By default the orientation of the chart is vertical. If the orientation of the chart is change to horizontal the domain axis will be the vertical line of the chart. 

 

Figure 5         Domain axis - Vertical orientation

 

 

Figure 6         Domain axis – Horizontal orientation

 

 

Range Axis (Y Axis)

The range axis (y axis) is the vertical line of the chart and contains the numerical values (counts).

 

Note: By default the orientation of the chart is vertical. If the orientation of the chart is change to horizontal the range axis will be the horizontal line of the chart. 

 

Figure 7         Range axis - Vertical orientation

 

 

Figure 8         Range axis – Horizontal orientation
