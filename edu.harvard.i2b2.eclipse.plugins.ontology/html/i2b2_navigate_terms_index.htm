i2b2 Navigate Terms View

The Navigate Terms View is designed to display the i2b2 vocabulary concepts that are maintained in the Ontology Management Cell of the i2b2 Hive. These concepts are used in many of the i2b2 views, such as Query Tool, Timeline and Export Data.

Introduction

This section describes the layout and contents of this view.

 

Layout of the Navigate Terms View

The Navigate Terms view displays the terms listed in the metadata table(s).  These terms are grouped (categorized) with other similar items.  These groupings are displayed in a hierarchal tree that can be expanded to view all the terms and modifiers in the associated category.

 

 

 

As stated above, the Navigate Terms view is hierarchical display of the terms and modifiers in the metadata table(s).  The modifiers will always appear with their associated concept.

There are several main branches or components of the tree that displays.

 

Table Name

The highest level is the name of the metadata table that stores the vocabulary. This level cannot be used when creating a query.

 

Containers

A container () has other folders, concepts, and / or modifiers.  It can be expanded and collapsed by clicking on the plus () or minus sign () next to it.  It can not be dragged to other views.

 

Modifier Containers

A modifier container () has other folders, concepts, and / or modifiers.  It can be expanded and collapsed by clicking on the plus () or minus sign () next to it.  It can not be dragged to other views.

 

Folders

A folder () contains other folders and/or concepts.  It can be expanded and collapsed by clicking on the plus () or minus sign () next to it.  It can also be dragged to other views.

 

Modifier Folders

A modifier folder () contains other modifier folders and/or modifiers.  It can be expanded and collapsed by clicking on the plus () or minus sign () next to it.  It can also be dragged to other views.

 

Concepts

The concept () is the lowest level of the hierarchy and cannot be expanded further. It can be dragged to other views.

 

Note: If a concept has modifiers associated the concept icon is replaced with the folder icon and the concept can be expanded to view the modifiers.

 

Modifiers

The modifier () is the lowest level of the hierarchy and cannot be expanded further. It can be dragged to other views.

 

 

Navigation

 

Expand/Collapse a Folder

Users can expand or collapse a folder by clicking on the plus sign () or negative sign () next to the folder.

 

Expand (Open) a Folder

1.       Click on the plus sign () next to the folder in Navigate Terms view.

2.       The folder will open and all the items in the folder will display.

 

Collapse (Close) a Folder

1.       Click on the negative sign () next to the folder in Navigate Terms view.

2.       The folder will close and all the items in the folder will no longer appear in the view.

 

 

Display Options

Users can define how many concepts to display and whether or not to show hidden terms and/or synonyms.  The display options are set in the Navigate Terms Options Dialog window, which can be accessed by clicking on the Options button ().

 

 

 

Maximum to Display

The Maximum number of children to display is used when the system is retrieving the items to be displayed when expanding the folder in Navigate Terms View.  Once the maximum number is reached a message will appear informing you that you have reached the maximum defined and that continuing to retrieve all the items (“nodes”) in the list may affect performance. You will have the option to continue or to cancel the action.

 

 

 

Show Hiddens

In the metadata table, an attribute is set to determine whether or not a term is active, inactive or hidden. If an item is marked as hidden it will not appear in the list of terms.  Selecting Show Hiddens will display those items marked to be hidden. For easy identification, hidden terms will be red when displayed in the hierarchical tree.

 

Show Synonyms

Some terms may be flagged in the metadata table as a synonym of another term.  Selecting Show Synonyms will display both the original term and the synonym in the Navigate Terms View. For easy identification, synonymous terms will be blue when displayed in the hierarchical tree.

 

Show Patient Counts

The Show patient counts option will display the total number of patients that have an observation fact entered for that item.

 

Show Short Tooltips

By default, the tooltip that displays when you hover over an item will contain the entire path name and the tooltip for that specific item. Enabling the Show short tooltips option will only display the tooltip information; the path will not be included.

 

A checkmark next to Use Short Tooltips means this option is on.  The tooltip will display only the name of the concept and not the entire path.

Example:

Acute myocardial infarction, unspecified site, initial episode of care

 

No checkmark next to Use Short Tooltips means this option is off and the tooltips will continue to display the path and name of the concept.

Example:

Diagnoses\...\Acute myocardial infarction\Acute myocardial infarction, unspecified site\Acute myocardial infarction, unspecified site, initial episode of care

 

Show Concept Code In Tooltip

In addition to displaying the tooltip, this option will also display the concept_cd associated to the item.

A checkmark next to Show Concept Codes in Tooltips means this option is on.  The tooltip will display the name of the concept and its associated code.

Example:

Acute myocardial infarction, unspecified site, initial episode of care (ICD9:410.91)

 

 

Disable Display Of Modifiers

The ability to search for modifiers will not be available if the Disable display of modifiers option is checked off.

 

 

 

Using Terms

This section describes how to use a concept or group of concepts.

 

Draggable Items

Individual as well as groups of concepts and modifiers can be dragged and dropped (copied) from the Find Terms view to other views in the i2b2 Workbench.

 

1.       Folder

A folder () and all its concepts (children) can be added to other views by dragging the folder name from Navigate Terms to one of the following views.

 

§         Query Tool View; drop in the panel

§         Timeline View; Create model for Timeline Tab

§         Workplace View; drop in a user or shared folder

 

 

2.       Modifier Folder

A modifier folder () and all its modifiers (children) can be added to other views by dragging the folder name from Navigate Terms to one of the following views.

 

§         Query Tool View; drop in the panel

§         Timeline View; Create model for Timeline Tab

§         Workplace View; drop in a user or shared folder

 

 

3.       Concept

A concept () can be added to other views by dragging the name of the concept from Navigate Terms to one of the following views.

 

§         Query Tool View; drop in the panel

§         Timeline View; Create model for Timeline Tab

§         Workplace View; drop in a user or shared folder

 

 

4.       Modifier

A modifier () can be added to other views by dragging the name of the modifier from Navigate Terms to one of the following views.

 

§         Query Tool View; drop in the panel

§         Timeline View; Create model for Timeline Tab

§         Workplace View; drop in a user or shared folder

 

 

Add to Query Tool View

A folder or individual concept can be added to the query tool view by simply dragging the item from Find Terms View and dropping it into one of the panels in the Query Tool View.

 

Add Concept / Folder To Query Tool View

1.       Highlight the concept () or folder () by clicking on its name.

2.       While holding the left mouse button down, drag the item over to the Query Tool View.

3.       Drop the item into the panel labeled Group 1.

 

 

 

4.       The item will now display in the panel for Group 1.

 

 

 

Add Modifier / Modifier Folder To Query Tool View

1.       Highlight the modifier () or modifier folder () by clicking on its name.

2.       While holding the left mouse button down, drag the item over to the Query Tool View.

3.       Drop the item into the panel labeled Group 1.

 

 

 

4.       The name of the concept associated to the modifier plus the modifier will now display in the panel for Group 1.

 

 

 

 

Add to Workplace View

Information in the workplace is related to the most common concepts and queries that an individual uses and in essence becomes their personal workplace.

 

 

 

Add Concept / Folder To Workplace View

1.       Highlight the concept () or folder () by clicking on its name.

2.       While holding the left mouse button down, drag the folder over to the Workplace view.

3.       Drop the item into either your workplace folder or the shared folder.

4.       The folder will now display in the Workplace view with the same name.

 

Add Modifier / Modifier Folder To Query Tool View

1.       Highlight the modifier () or modifier folder () by clicking on its name.

2.       While holding the left mouse button down, drag the item over to the Workplace View.

3.       Drop the item into either your workplace folder or the shared folder.

4.       The name of the concept associated to the modifier plus the modifier will now display in the Workplace view with the same name.

 

 

 

Add to Timeline View

The timeline view displays the information in a graphical representation of a chronological sequence of events.

 

 

 

Add Concept / Folder To Timeline View

1.       Highlight the concept () or folder () by clicking on its name.

2.       While holding the left mouse button down, drag the item over to the Timeline View.

3.       Drop the item into the concept list section of the Create model for Timeline tab.

4.       The item will now display in the Timeline View.

 

 

Add Modifier / Modifier Folder To Timeline View

1.       Highlight the modifier () or modifier folder () by clicking on its name.

2.       While holding the left mouse button down, drag the item over to the Timeline View.

3.       Drop the item into the concept list section of the Create model for Timeline tab.

4.       The name of the concept associated to the modifier plus the modifier will now display in the Timeline View.
