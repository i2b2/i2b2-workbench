Add to Workplace View

Information in the workplace is related to the most common concepts and queries that an individual uses and in essence becomes their personal workplace.

 

 

 

Add Concept / Folder To Workplace View

1.       Highlight the concept () or folder () by clicking on its name.

2.       While holding the left mouse button down, drag the folder over to the Workplace view.

3.       Drop the item into either your workplace folder or the shared folder.

4.       The folder will now display in the Workplace view with the same name.

 

Add Modifier / Modifier Folder To Query Tool View

1.       Highlight the modifier () or modifier folder () by clicking on its name.

2.       While holding the left mouse button down, drag the item over to the Workplace View.

3.       Drop the item into either your workplace folder or the shared folder.

4.       The name of the concept associated to the modifier plus the modifier will now display in the Workplace view with the same name.
